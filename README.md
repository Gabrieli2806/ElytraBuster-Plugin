---

Transforms Minecraft gameplay by adding a set of four specialized TNT variants that seek out Elytra-gliding players or crossbow-launched fireworks within a 50-block radius. Bringing strategic depth to servers.

---

**Features**

*Escalating Challenges*
* 4 TNT Variants - Ranging from Level 1 (basic tracking, 2.0 explosion power) to Level 4 (contact explosion, 4.0 power).
* Smart Tracking - Actively seeks out Elytra-gliding players or crossbow-launched fireworks within a massive 50-block radius.

*Deployment & Crafting*
* Intuitive Recipes - Craftable through intuitive recipes that are automatically unlocked in the recipe book when players join the server.
* Dispenser Support - Fully deployable via dispensers, perfect for building automated anti-air traps and base defenses.

*Immersive Mechanics*
* Dynamic Ascent - Features custom ascent mechanics to chase down flying targets in the sky.
* Audio & Visual Cues - Includes musical note sounds and target warnings to alert players of incoming threats.
* Custom Death Messages - Unique broadcast messages when a player is successfully taken down by an ElytraBuster.

*Admin Tools*
* Spawn Command - Instantly accessible via the `/elytrabustertnt <1-4>` command for quick deployment or event management.
