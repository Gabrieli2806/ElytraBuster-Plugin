# ElytraBuster

Special TNT with 4 levels of increasing speed and a special level that explodes on contact. Rises 20 blocks, floats, chases elytra players persistently with a title warning, and explodes after 15 seconds with musical notes. Craftable and obtainable via command for OPs.

![Platform](https://img.shields.io/badge/platform-PaperMC%20%2F%20Bukkit-blue) ![API](https://img.shields.io/badge/api--version-1.21-green)

## Commands

| Command | Description | Permission |
|---------|-------------|------------|
| `/<command> <level>` | Gives the player a special ElytraBuster TNT of the specified level (1-4). | `elytrabuster.tnt` |

## How to Use

1. Download the `.jar` from [Releases](https://github.com/Gabrieli2806/ElytraBuster-Plugin/releases).
2. Place it inside your server's `plugins/` folder.
3. Restart or reload the server.
4. Use the commands listed above in-game or from the console.
